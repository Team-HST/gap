# Gaze Analysis Program(GAP)

## 개요
```
Gaze Analysis Program
```

## 기술스택
```
Frontend: Vue.js + Electron
```

## 실행방법
#### 1. 웹으로 실행
```
$ git clone https://github.com/Team-HST/gap
$ cd gap
$ npm install
$ npm run serve
```

#### 2. 데스크톱 앱으로 실행
```
$ git clone https://github.com/Team-HST/gap
$ cd gap
$ npm install
$ npm run electron:serve
```
